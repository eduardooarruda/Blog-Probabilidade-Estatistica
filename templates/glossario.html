Glossário

A

Amostra: Uma amostra é um conjunto de dados coletados e/ou selecionados de uma população estatística por um procedimento definido. É muito utilizado quando o conjunto de dados é muito grande.

C

Classes: São intervalos numericos criados para facilitar o estudo de um conjunto de dados.

D

Dado Estatístico: É um elemento que pertence ao conjuto de dados de uma pesquisa que é chamada de população.

P

População: São todos os elementos (dados) que pertecem a uma pesquisa.

R

Rol: é uma forma organizar os dados quantitativos em ordem crescente ou decrecente.

T

Tabela de distribuição de frequência: É uma tabela que mostra a frequência que cada dado aparece no conjunto de dados.

V

Variável: É a característica dos elementos da pesquisa. Pode ser dividido em variaveis qualitativas e quantitativas.
